## 数据流

```
用户输入
  │
  ▼
┌─────────┐
│ Claude  │  SessionStart: session-init → superboost → bd prime
│  Code   │  (环境 + 自检 + 记忆 + 待办 全部注入)
│ (Opus)  │
└────┬────┘
     │
     ├─── 读文件 ──→ MCP/lean-ctx ctx_read    ← 缓存命中 ~13 tokens
     │                                        ← 10 种模式按需
     │
     ├─── 搜代码 ──→ MCP/fff ffgrep/fffind    ← Frecency 排序
     │              → MCP/lean-ctx ctx_search  ← 压缩结果
     │
     ├─── 执行命令 → PreToolUse → rtk hook     ← 40+ 命令过滤
     │              → MCP/lean-ctx ctx_shell   ← 95+ 压缩模式
     │
     ├─── 压缩内容 → MCP/headroom compress     ← 60-95% 节省
     │
     ├─── 任务管理 → CLI bd create/update/close ← Dolt 数据库
     │
     ├─── 派 Agent → PreToolUse resource-guard ← 配额检查
     │              → VoltAgent 子 Agent       ← 7 个专业 Agent
     │
     └─── 记忆归档 → Plugin claude-mem         ← 跨 session
                    → CLI bd remember          ← 项目级知识
```

## 压缩三件套分工

```
命令输出                   文件内容                API/对话
    │                         │                      │
    ▼                         ▼                      ▼
  RTK                      lean-ctx               headroom
PreToolUse Hook           MCP Server             MCP Server
(自动, 无感)              (Agent 主动调)          (Agent 主动调)
git/diff/npm/test...      10 种 read mode         SmartCrusher(JSON)
40+ 命令定制              缓存命中 ~13 tokens      CodeCompressor(AST)
                          安全检查                Kompress-base(文本)
                                                  CCR 可还原
```

```

──────                              ──────
planning-with-files                 beads (bd)
                                    
task_plan.md    ──→  全文本读取     bd prime       ──→ ~1-2k tokens 注入
findings.md     ──→  全文本读取     bd ready       ──→ 只看未阻塞任务
```
